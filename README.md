# Interactive Mini Game

Simple game built with Angular 20.

Live Demo:
https://helga-welt.github.io/interactive-mini-game/

## Features

- Angular 20 Standalone Components
- Signals
- inject()
- input.required()
- output()
- Modern Control Flow (@if, @for)
- Responsive Design
- Glassmorphism UI

## Technologies

- Angular 20
- TypeScript
- SCSS

## Installation

npm install

## Run

ng serve

Open http://localhost:4200

## Build

ng build

## Project Structure

src/
├── app/
│   ├── components/
